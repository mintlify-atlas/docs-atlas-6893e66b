# Welcome to ImageGlass

ImageGlass is a lightweight, open-source image viewer designed for seamless viewing of images in a clean and intuitive interface. Built for Windows 10 and 11, it offers a perfect balance of simplicity and power for both everyday users and professionals.

## What is ImageGlass?

ImageGlass provides a fast, efficient way to view and manage your image files with minimal system overhead. Whether you're a photographer reviewing shots, a designer checking assets, or simply browsing personal photos, ImageGlass delivers a smooth viewing experience with advanced features when you need them.

<Note>
  ImageGlass is free and open-source software licensed under GPLv3. It's free for both personal and commercial use.
</Note>

## Why Choose ImageGlass?

### Comprehensive Format Support

ImageGlass supports an extensive range of image formats out of the box:

- **Common formats**: JPG, PNG, GIF, BMP, TIFF, ICO, SVG, WEBP
- **Modern formats**: AVIF, JXL (JPEG XL), HEIC/HEIF
- **RAW camera formats**: CR2, CR3, NEF, ARW, DNG, RAF, ORF, and many more
- **Specialized formats**: PSD, FITS, HDR, EXR, FLIF, QOI
- **Animated images**: APNG, animated GIF, animated WEBP

### Designed for Workflow Efficiency

ImageGlass includes features that enhance your productivity:

- **Explorer sort order** - Maintains the same file order as Windows Explorer
- **Quick setup wizard** - Get started quickly with guided configuration
- **File associations** - Easily set ImageGlass as your default image viewer
- **Customizable toolbar** - Access your most-used functions quickly
- **Auto-update** - Stay current with the latest features and improvements (Classic version)

### Open Source and Community-Driven

ImageGlass is developed openly on GitHub with contributions from developers worldwide. The project values:

- **Transparency** - All code is publicly available
- **Community feedback** - Feature requests and bug reports drive development
- **Privacy** - No telemetry or tracking
- **Extensibility** - Support for plugins and custom tools

## System Requirements

Before installing, ensure your system meets these requirements:

- Windows 10/11 64-bit, version 1809 (build 17763) or later
- Optional: WebView2 Runtime for advanced features

<Warning>
  ImageGlass requires a 64-bit version of Windows. 32-bit systems are not supported.
</Warning>
